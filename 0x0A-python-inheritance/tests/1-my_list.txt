
A simple `doctest` for `1-my_list` module.

>>> MyList = __import__('1-my_list').MyList

>>> my_list = MyList([4, 2, 3, 1])

>>> my_list.print_sorted()
[1, 2, 3, 4]

>>> MyList([3, 2, 1, 1]).print_sorted()
[1, 1, 2, 3]

>>> MyList(['hello', 'just', 'checking']).print_sorted()
['checking', 'hello', 'just']

>>> MyList(('hello', 'just', 'checking')).print_sorted()
['checking', 'hello', 'just']

>>> MyList({1: 'a', 2: '3'}).print_sorted()
[1, 2]

>>> MyList({1: 'a', 2: '3'})
[1, 2]

>>> MyList([None, None])
[None, None]
